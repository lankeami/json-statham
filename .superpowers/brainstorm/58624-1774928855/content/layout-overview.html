Key Elements

  • Top bar: jq query input, copy button, theme toggle (moon/sun icon)
  • Left panel: Textarea for raw JSON input (paste anything, any format)
  • Right panel: Syntax-highlighted, collapsible tree output
  • "Format & Query" button: Parses, pretty-prints, and applies jq filter
  • Status bar: Validation status, key count, byte size, jq result count
  • Collapsed nodes: Show summary like { ... 2 keys } or [ ... 5 items ]
